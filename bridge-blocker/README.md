# 堵桥模拟器

一个父子 vibe coding 的小项目，灵感来自《三角洲行动》的"堵桥"玩法。

## 怎么打开

直接双击 `index.html`，浏览器里就能玩。
没有任何构建步骤、没有依赖。整个游戏（包括三套战士贴图）都在这一个 HTML 文件里（base64 内嵌）。

## 怎么玩

- 打开页面看到开始菜单，**点绿色"▶ 开始游戏"按钮** 才开局
- **鼠标点击屏幕任意位置** → 乌鲁鲁朝那里开枪
- **鼠标点蜂医身上** → 释放烟雾技能（不开枪）
- **R 键** → GAME OVER 后重开
- 调节面板（画面下方）：
  - 拖滑杆改"爆头范围"
  - 数字框直接输精确值
  - 勾选"显示爆头区"在敌人身上看见红框
  - 调"每位最多几个"控制刷新压力

## 角色

| 角色 | 阵营 | 位置 | HP | 武器 / 技能 |
|---|---|---|---|---|
| 乌鲁鲁 | 玩家 | 桥南 | 200 | 鼠标控制射击 |
| 威龙 | 队友（AI） | 桥北 | 200 | 每 2 秒一枪，50% 命中（强制爆头） |
| 蜂医 | 奶妈（AI） | 桥中央 | — | 每 10 秒给乌鲁鲁+威龙各 +25 HP；点击触发烟雾技能（CD 40s / 持续 20s） |
| 步枪手 | 敌方 | 桥右掩体 | 100 | 每 3 秒一枪，25% 爆头(50) / 25% 身体(25) / 50% 空枪 |
| 狙击手 | 敌方 | 桥右掩体 | 100 | 每 3 秒一枪，25% 爆头(100) / 25% 身体(50) / 50% 空枪 |

每三个新刷的敌人里 **1 个狙击手 + 2 个步枪手**。狙击手头顶有 🎯 标记。

## 战场结构

```
画面 640×800（俯视 2D）

   桥北（威龙队友）
      ┃
      ┃        ╱── 三个掩体（三角阵）
      ┃ 桥    ┃ ─→  敌人从右边冲来，到位后随机上下晃动
      ┃        ╲──
      ┃
   桥南（乌鲁鲁玩家）
```

- 桥右每 7 秒刷新一波敌人，每个终点位最多 2 个（可调）
- 三个终点位独立计数，满了就跳过该位置
- 敌人 HP 归 0 → 半透明躺地，5 秒后从画面与计数中清除
